# Changelog

## v3.4.1 (2026-07-08)

与 v3.4.0 相同，版本号修正以确保插件市场正确推送。

## v3.4.0 (2026-07-08)

### 🐛 Fixed

- **@提及解析修复：** 修复了复制/手动输入 `@某人` 时无法识别的问题。现在会通过群成员列表查询昵称和群名片来解析 `@` 提及，不再只能识别 QQ 选择器生成的 @。
- **V2 双人效果顺序修复：** 修复了 `@A @B` 与 `@B @A` 生成同一张图的问题（重构了 QQ 提取逻辑，现在严格按 @mention 在文本中的出现顺序排列）。
- **V1/V2 效果混入修复：** 修复了在线同步时 V2 效果被混入 V1 菜单的问题（现在同步时按已知 V2 key 分流，分别更新两个菜单）。
- **指令路由修复：** 修复了 `/表情更新` 命令被主表情命令劫持的问题（`\s*` 改为 `\s+`，要求参数前必须有空格）。

### ✨ Added

- 新增群成员列表缓存（5 分钟 TTL），避免频繁查询群成员 API。
- 新增 `api.call` 插件能力声明，支持跨插件 API 调用。
- `/表情` 无参数时显示帮助信息。

### 🔧 Changed

- 版本号升级至 3.4.1。
- 主命令正则 `\s*` 改为 `\s+`，避免 `/表情更新` 等子命令被劫持。
